# 🔗 URL Shortener (TypeScript + Node.js)

A simple and efficient URL shortener built using **TypeScript**, **Node.js**, and **Express**.
It converts long URLs into short links and redirects users to the original URL.

---

## 🌍 Live Demo

👉 https://url-shortner-ji0b.onrender.com/

---

## ✨ Features

* 🔗 Shorten long URLs
* 🚀 Instant redirection
* 🌐 Simple frontend UI
* ⚡ Fast and lightweight
* 🧠 Built with TypeScript (type-safe)

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* TypeScript
* HTML + JavaScript (Frontend)
* Render (Deployment)

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/url-shortener.git
cd url-shortener
```

---

### 2. Install dependencies

```bash
npm install
```

---

### 3. Run in development

```bash
npm run dev
```

---

### 4. Build project

```bash
npm run build
```

---

### 5. Run production

```bash
npm start
```

---

## 🌐 Usage

1. Open the app in browser
2. Enter a long URL
3. Click **Shorten**
4. Use generated short link

---

## ⚠️ Limitations

* Uses in-memory storage (data resets on server restart)
* No analytics or tracking yet
